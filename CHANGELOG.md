# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.2] - 2026-04-10

### Added
- Display Antom configuration in "Recommended Solutions" group in admin
- Add Antom logo to admin configuration
- Add custom Fieldset with Configure/Close button functionality

### Changed
- Update admin configuration UI to match Airwallex style

## [1.0.1] - 2024-04-04

### Fixed
- Move resource tag from group to section level in system.xml

## [1.0.0] - 2024-04-04

### Added
- Initial release.
- Support for Credit/Debit Cards via Antom.
- Support for Google Pay and Apple Pay.
- Integration with Hyvä Checkout.
- CSP whitelist for payment scripts.
